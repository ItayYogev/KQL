# Search for Software on PC
This query searches for a specific software application installed on a designated PC.

The query is valuable for security analysts as it helps identify software installations and their versions on specific devices within the organization. By monitoring software usage, analysts can:

- Detect outdated or vulnerable software that may pose security risks.
- Ensure compliance with licensing agreements and software policies.
- Correlate software presence with security incidents, enabling better threat assessment and response.
- Identify unauthorized software installations that could indicate potential security breaches or non-compliance with organizational policies.
